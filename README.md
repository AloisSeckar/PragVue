# PragVue
Website for Vue.js conference in Prague

## nuxt-ignis
This project uses [Nuxt Ignis](https://github.com/AloisSeckar/nuxt-ignis) to simplify dependency management.

To make it run locally, you need to provide `.env` file with expected [configuration](https://nuxt-ignis.netlify.app/2-1-configuration.html). Rename provided `.env.example` to `.env` to get started.
